Basic Replace

Simple replace of two variables that have been defined 
{title}: {subtitle}
{!}
{title}: {subtitle}

If/Else with Value

Using a predefined value with an if/else statement 
Value: {true} 
Tested: 
{if:true}
	True
{else}
	Not True
{end}

{!}Value: {true}

{!}{if:true}
{!}	True
{!}{else}
{!}	Not True
{!}{end}

If/Else with Boolean

Using a boolen calculation in the HTML for evaluation 
Tested: 
{if:100>0}
	True
{else}
	False
{end}

Tested: 
{!}{if:100>0}
{!}	True
{!}{else}
{!}	False
{!}{end}

Switch

Switch upon a predefined variable, comparing with provided strings 
Value: {name} 
Tested:
{switch:name}
{case:"dave"} Was dave
{case:"john"} Was john
{case:"steve"} Was steve
{endswitch}

{!}Value: {name}
{!}Tested:
{!}{switch:name}
{!}	{case:"dave"} Was dave
{!}	{case:"john"} Was john
{!}	{case:"steve"} Was steve
{!}{endswitch}

ForEach

Loop over a predefined array, outputting each value 
Value: {people} 
Tested:{foreach:people,value} 
	{value}
{end} 

{!}Value: {people}
{!}Tested:
{!}{foreach:people,value} 
{!}	{value}
{!}{end}

ForEachKey

Loop over a predefined array, outputting each value and key 
Value: {people} 
Tested:{foreach:people,i,value} 
	{i} {value}
{end}

{!}{foreach:people,i,value} 
{!}	{i} {value}
{!}{end}

Ignore

Tell the Parser to ignore the variables in the line 
{!}{ignored}

{!}{!}{ignored}

Arrays

Access a predefined array via index 
{people[0]}

{!}{people[0]}

Objects

Access an predefined objects function 
{person.fullName()}

{!}{person.fullName()}
